## Introduction

Email Extractor Pro is designed to simplify the process of extracting email addresses from large texts. It uses a powerful regex pattern to identify and extract emails, making it a valuable tool for marketing, research, or any scenario where email extraction is necessary.

## Features

- **Email Extraction**: Extracts email addresses from any text input.
- **Unique Email Filtering**: Automatically removes duplicate emails.
- **User-Friendly Interface**: Easy to use for both technical and non-technical users.
- **Local Processing**: All data is processed locally in your browser, ensuring privacy.

## How to Use

1. **Paste Text**: Copy and paste the text containing email addresses into the input field.
2. **Extract Emails**: Click the "Extract Emails" button to process the text.
3. **View Results**: Extracted emails will be displayed below, along with options to copy or download them.

## Installation

This project is a Flask web application. To run it locally:

1. Clone the repository using `git clone https://github.com/yourusername/email-extractor.git`.
2. Install required packages with `pip install -r requirements.txt`.
3. Run the application using `python app.py`.
